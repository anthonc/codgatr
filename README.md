**CodGatr** is the essential extension for developers working with AI. It acts as a bridge between your AI conversations and your IDE, allowing you to instantly extract, organize, and download code blocks generated by platforms like ChatGPT, Claude, and Gemini.

Forget manually copying and pasting files one by one. CodGatr intelligently detects identifying comments (like `# filename: script.py`) to name your files automatically, bundles everything into a clean ZIP archive, or lets you download individual snippets with a single click.

Now featuring a **Premium "Cyber-Sleek" Interface** with dark glassmorphism, smooth animations, and a focus on developer experience.

## ğŸš€ Installation (Manual)

While we are reviewing our submission to the Chrome Web Store, you can install the developer version today:

1. **Download the Source**:
   - Clone this repo: `git clone https://github.com/anthonc/codgatr.git`
   - Or download the ZIP from the "Code" button above.

2. **Load in Chrome**:
   - Open `chrome://extensions/`
   - Enable **Developer mode** (top right toggle).
   - Click **Load unpacked**.
   - Select the `codgatr` folder.
